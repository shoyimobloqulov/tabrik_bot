# New Year Greeting Image Bot

A Python project for generating customized New Year greeting images with text and emoji overlays. Users can test templates locally and generate images with adjustable font sizes and positions.

---

## Features

- Customizable New Year image templates.
- Add text and emojis to images.
- Adjustable font sizes and text positions.
- Test image generation using `test.py`.
- Easy deployment with `Procfile` for platforms like Heroku.

---

## Folder Structure
app/ # Core application logic
core/ # Scripts for image generation and overlay
out/ # Generated images output
static/templates/ # Image templates
main.py # Main script for image generation
test.py # Script to test image generation
requirements.txt # Python dependencies
Procfile # Deployment file

```bash

cd your-repo-name
```

## Install dependencies
```bash
pip install -r requirements.txt
```

# Usage
## 1. Run the main script
```bash
python main.py
```
<i>This will generate a New Year greeting image using the default template and settings.</i>
## 2. Test image generation
```bash
python test.py
```
<i>test.py allows you to preview the templates and test text/emoji overlays before full usage.</i>
